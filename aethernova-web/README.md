# Frontend Readme

## AetherNova Frontend

Современное веб-приложение для управления AI агентами, построенное на React + TypeScript + Vite.

### 🚀 Технологический стек

- **React 18** - библиотека пользовательского интерфейса
- **TypeScript** - типизированный JavaScript
- **Vite** - быстрый инструмент сборки
- **TailwindCSS** - утилитарный CSS фреймворк
- **Radix UI** - примитивы пользовательского интерфейса
- **Zustand** - управление состоянием
- **React Query** - кеширование и синхронизация данных
- **Recharts** - библиотека для графиков и диаграмм

### 📋 Возможности

- **Дашборд агентов** - мониторинг статуса и производительности AI агентов
- **Управление задачами** - создание, редактирование и отслеживание задач
- **Системная аналитика** - визуализация метрик и производительности
- **Real-time обновления** - WebSocket интеграция для live данных
- **Адаптивный дизайн** - оптимизация для всех устройств
- **Темная/светлая тема** - переключение тем интерфейса

### 🏗️ Структура проекта

```
aethernova-web/
├── src/
│   ├── components/          # React компоненты
│   │   ├── ui/             # UI примитивы
│   │   └── layout/         # Layout компоненты
│   ├── pages/              # Страницы приложения
│   ├── lib/                # Утилиты и библиотеки
│   │   ├── api.ts          # API клиент
│   │   └── websocket.ts    # WebSocket клиент
│   ├── store/              # Управление состоянием
│   └── styles/             # Стили и темы
├── public/                 # Статические файлы
├── Dockerfile             # Docker конфигурация
├── nginx.conf             # Nginx конфигурация
└── package.json           # Зависимости проекта
```

### 🚀 Запуск разработки

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build

# Предварительный просмотр сборки
npm run preview
```

### 🐳 Docker

```bash
# Сборка Docker образа
docker build -t aethernova-frontend .

# Запуск контейнера
docker run -p 3000:80 aethernova-frontend
```

### 🔧 Интеграция с Backend

Frontend настроен для работы с FastAPI backend:

- **API Base URL**: `http://localhost:8000/api`
- **WebSocket URL**: `ws://localhost:8000/ws`
- **Прокси конфигурация** в Vite для разработки

### 📊 Основные страницы

1. **Dashboard** (`/dashboard`) - Главная страница с обзором системы
2. **Agents** (`/agents`) - Управление AI агентами
3. **Tasks** (`/tasks`) - Мониторинг и управление задачами
4. **Analytics** (`/analytics`) - Аналитика и метрики
