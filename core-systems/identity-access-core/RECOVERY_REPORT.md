### 1. Архитектурное восстановление

#### ✅ Создан полноценный сервис аутентификации (`src/authentication.py`):
- **Хеширование паролей** с bcrypt (cost factor: 12)
- **JWT токены** для API доступа

#### ✅ Создан движок авторизации (`src/authorization.py`):
- **RBAC** (Role-Based Access Control)
- **ACL** для ресурсов (Access Control Lists)
- **Гибкие политики** доступа
- **Предустановленные роли**: superuser, admin, emergency, user, readonly, guest
- **Управление разрешениями** на уровне пользователей и ролей

**Код**: 280+ строк, полностью async
