# human-sovereignty-core

**Human Sovereignty Core** — фундаментальный модуль Aethernova, обеспечивающий абсолютный приоритет человека над ИИ-системами, агентами и автоматизированными контурами управления.

Модуль фиксирует и технически гарантирует, что:
- окончательные решения принадлежат человеку;
- ИИ не может навязать цели, действия или интерпретации;
- любое автоматическое поведение подчиняется явному человеческому мандату.

---

## Назначение

Human Sovereignty Core решает ключевую проблему AGI-подобных систем:  
**координация и контроль не могут быть автономными без человеческого суверена.**

Модуль внедряется как обязательный слой между:
- coordination / planning слоями,
- agent-mesh и LLM,
- execution / actuation уровнями.

---

## Базовые принципы

1. **Human-in-the-Loop по умолчанию**  
   Любое критичное решение требует подтверждения человека.

2. **Human-in-Command**  
   Человек может в любой момент:
   - отменить решение,
   - изменить цель,
   - остановить систему.

3. **No Silent Autonomy**  
   Запрещены:
   - скрытые цели,
   - самопереопределение приоритетов,
   - неявная эскалация полномочий ИИ.

4. **Explainability First**  
   Любое предложение ИИ обязано быть:
   - объяснимым,
   - трассируемым,
   - проверяемым.

---

## Роль в архитектуре Aethernova

Human Sovereignty Core взаимодействует с:

- `genius-core` — проверка намерений агентов  
- `decision-core` — валидация решений  
- `policy-core` — соответствие правилам человека  
- `ethics-core` — запрет античеловеческих стратегий  
- `zero-trust-core` — запрет неавторизованных действий  

Логически располагается **выше coordination layer**, но **ниже человеческого интерфейса управления**.

---

## Типовые сценарии

- Блокировка автономного решения, противоречащего воле оператора.
- Принудительный rollback действий агента.
- Требование ручного подтверждения при:
  - изменении стратегии,
  - перераспределении ресурсов,
  - выходе за заданные рамки.

---

## Гарантии

Модуль гарантирует:
- невозможность полного AGI-контроля без человека;
- невозможность «дрейфа целей»;
- невозможность скрытого самоуправления системой.

---
