## 1. Цели системы
CSMarket это маркетплейс для листинга, поиска, покупки и продажи игровых предметов CS2 с интеграцией Telegram и веб-интерфейса, с оплатой вне Steam.

Цели
1. Надежность и предсказуемость: контроль состояния заказов и платежей, идемпотентность операций.
2. Безопасность: защита аккаунтов, ключей, админ-доступов, антифрод и аудит.
3. Масштабируемость: горизонтальное масштабирование API, фоновых воркеров, поискового слоя.
4. Наблюдаемость: трассировки, метрики, логи, аудит событий.
5. Совместимость: единый формат событий и контрактов между сервисами.

## 2. Контекст и границы домена
CSMarket работает как посредник
1. Пользовательский интерфейс: Telegram бот и Web UI.
2. Ядро домена: листинги, ордера, сделки, выплаты, спорные ситуации.
3. Интеграции: платежные провайдеры и источники рыночных цен.

Принцип: внешние интеграции считаются недоверенными. Все входящие вебхуки валидируются, подписываются и проходят идемпотентную обработку.

## 3. Высокоуровневая схема
Компоненты
1. Edge слой: gateway-core (rate limit, WAF правила, JWT валидация, маршрутизация).
2. API слой: csmarket-api (FastAPI, async SQLAlchemy).
3. Background слой: csmarket-worker (очереди, outbox relay, синхронизация статусов).
4. Data слой: Postgres, Redis, объектное хранилище, поисковый индекс.
5. Observability: метрики, логи, трассировки.
6. Security: policy, rbac, audit, secrets.
