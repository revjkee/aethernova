# Archived roadmap overlays

This directory preserves roadmap snapshots whose component names collided with
canonical systems under `core-systems/`.

The snapshots are not runtime code and must not be imported or deployed. They
contain a mixture of exact copies and experimental deltas recovered on
2026-07-16. Review unique files against the corresponding canonical component
before intentionally porting any change. Git history remains the source for
the original paths.
