# Кодировка: UTF-8, текстовый файл, CR/LF допускается. Google игнорирует невалидные строки (например, BOM).
# Подтв.: Google robots.txt spec. (File format) — см. ссылку выше.

# БАЗОВАЯ ПОЛИТИКА: разрешаем краулинг всего (безопасно для SPA; robots.txt не управляет индексированием).
# Подтв.: Google Intro — robots.txt управляет доступом краулера, а не индексированием; для блокировки индексации используйте noindex/авторизацию.
User-agent: *
Allow: /

# ВАЖНО:
# 1) robots.txt — добровольный протокол, не механизм авторизации доступа.
#    Подтв.: RFC 9309 ("These rules are not a form of access authorization").
# 2) Максимальный размер файла, который парсеры обязаны обрабатывать — ≥ 500 KiB (Google придерживается 500 KiB).
#    Подтв.: Google robots.txt spec., RFC 9309.

# ЕСЛИ НУЖНО ОГРАНИЧИТЬ НАГРУЗКУ ДЛЯ ОТДЕЛЬНЫХ КРАУЛЕРОВ:

# Используйте с осторожностью и ТОЛЬКО при реальной необходимости.

# User-agent: bingbot
# Crawl-delay: 10
